Clase 13

02/05/2023

Hoy tocó la clase virtual, el profesor nos enseñó sobre el diagrama de flujo, en donde Se puede utilizar un rombo en el diagrama de flujo para representar la instrucción "if", donde la condición se escribe dentro del rombo. Los diagramas de flujo son útiles para planificar y entender la lógica de un programa antes de escribir el código y la instrucción "else" es una estructura de control de flujo que se utiliza en la programación para ejecutar un bloque de código si no se cumple la condición especificada en una instrucción "if". El uso del "else" permite especificar un bloque de código que se ejecutará en caso de que la condición en el "if" no se cumpla.

Clase 14

03/05/2023

Continuando con la anterior clase, en otras palabras, si la condición especificada en una instrucción "if" se cumple, se ejecuta un bloque de código, y si no se cumple, se ejecuta el bloque de código especificado en la instrucción "else", esto permite que el programa tome decisiones y realice diferentes acciones en función de las condiciones que se cumplan, En resumen, la instrucción "else" se utiliza para ampliar la funcionalidad de una instrucción "if" y proporcionar una lógica más avanzada a los programas, Además hicimos ejercicios en clase para poder facilitar nuestro aprendizaje.
